## Описание проекта

Travel Time — это современное веб-приложение для управления системой бронирования отелей. Приложение позволяет администраторам управлять отелями, номерами и бронированиями, а также предоставляет мощную систему поиска и фильтрации с учетом доступности номеров на выбранные даты.

### Ключевые особенности проекта:

- **Управление отелями и номерами** — полноценный CRUD функционал
- **Система бронирований** — календарное отображение с временной шкалой
- **Расширенная фильтрация** — многоуровневая система фильтров с синхронизацией URL
- **Виртуализация** — оптимизация производительности для больших списков
- **Экспорт данных** — экспорт списков отелей для мессенджеров
- **Аутентификация** — безопасная система входа и авторизации

**Ответ:** Да, формат работы обсуждается индивидуально. Главное — качество и своевременность выполнения задач.

**Ответ:** На данный момент проект ведется одним разработчиком. Возможен переход к командной работе в будущем.

### Вопрос: Какие есть возможности для роста?

- Работа с современным стеком
